Formato dgi - Localización nicaragüense

Emite 4 reportes que forman parte del Formato dgi - Localización nicaragüense

Funcionalidad: Una vez creada la factura de cliente o proveedor permite emitir los reportes de NOTA DE CRÉDITO, NOTA DE DÉBITO, FACTURA, cuando se realiza el pago de cliente o de proveedor permite crear el reporte de RECIBO DE CAJA.

Principales características:

  • Posibilidad de tener guadado los informes emitidos mediante una NOTA DE CRÉDITO, NOTA DE DÉBITO, FACTURA, RECIBO DE CAJA.
